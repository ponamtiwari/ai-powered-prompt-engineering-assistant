## What is this project?

**AI-Powered Prompt Engineering Assistant** is a client-side web application that transforms rough user inputs into clear, professional AI prompts. It includes real-time grammar correction, multilingual support, domain-aware templates, speech-to-text, and optional OpenAI integration.

Built with **React 18**, **TypeScript**, **Vite**, **Tailwind CSS**, and **OpenAI API**.

## Quick start

```bash
git clone https://github.com/ponamtiwari/ai-powered-prompt-engineering-assistant.git
cd ai-powered-prompt-engineering-assistant
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
