## Open the app

**Development:**
```bash
npm run dev
```
Open `http://localhost:5173`

**Requirements:** Modern browser, optional OpenAI API key

---

## Step 1: Enter your prompt

**Type** in the text area, or **click the microphone** for voice input.

Select speech language from dropdown, or **Auto** (requires API key for Whisper).

---

## Step 2: Add API key (optional)

Click **Add API Key** → paste OpenAI key → **Save**

Unlocks: AI enhancement, translation, Whisper, Generate Output

---

## Step 3: Review grammar

If suggestions appear:

- **Accept All Changes** — apply corrections
- **Keep Original** — dismiss

---

## Step 4: Enhance prompt

Click **Enhance Prompt** → wait for **Enhanced Prompt** section

---

## Step 5: Enhance more (optional)

Click **Enhance More** for a deeper, more detailed prompt.

---

## Step 6: Use the result

| Action | How |
|--------|-----|
| Copy | Click **Copy** |
| Download | **TXT** or **DOCX** |
| External AI | **Continue with AI** → pick platform |
| Real output | **Generate Output** (needs API key) |

---

## Two output types

| Section | What it is |
|---------|------------|
| **Generated Output** | Template preview (not from OpenAI) |
| **AI Generated Output** | Real content from OpenAI |

---

## Language notes

- English prompts work without API key
- Non-English enhancement requires API key for translation
